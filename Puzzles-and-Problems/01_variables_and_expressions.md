## Input; output; variables; arithmetic

> [!Note]
> At this point, you're still getting used to datatypes.
> You can always check the type of a variable using the `type()` function.
> Recall that `input()` always returns a string value.

> [!Reference]
> `input()`: Waits for the program user to input some text, finishes when they press "enter"
> `int()`: Turns a variable into an integer (or tries to)
> `str()`: Turns a variable into a string
> `float()`: Turns a variable into a floating-point number
> `print()`: Outputs text to the console
>
> String concatenation can be done by `+` adding strings together.
> Note well that string concatenation will fail if you mix datatypes. Consider the above functions as hints.
